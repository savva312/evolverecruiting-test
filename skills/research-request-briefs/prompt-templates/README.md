# EvoResearcher prompt templates

These are **copy/paste templates** for the `## Research prompt (EvoResearcher)` section of an EvoResearcher ticket.

## How to use
1. Pick a template below.
2. Replace placeholders like `<COUNTRY>`, `<SCHOOL_NAME>`, `<PROFILE_DIR>`, `<OUTPUT_FILES>`.
3. Delete sections you don’t need (keep it scoped to the ticket’s allowed write paths).
4. Default to `Research rounds: 5` unless the topic is very easy (1–3) or unusually hard (10–15).
5. Paste into the ticket under `## Research prompt (EvoResearcher)`.

## Templates
- `high-school-profile.md` — deep, comprehensive profiling of a high school for UNIC/UNIC Athens recruiting (writes a long-form research artifact report, then extracts a concise profile `README.md`).

## Adding new templates (later)
- Add a new file in this folder using kebab-case (e.g., `education-agent-profile.md`).
- Keep templates **generic** (no country-specific assumptions).
- Include: deliverables, evidence/citation rules, privacy guardrails, and a round plan if `Research rounds > 1`.
- Add the new template to the list above.

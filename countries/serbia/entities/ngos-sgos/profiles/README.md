# NGO/SGO profiles

Add one file per organization. Use consistent sections to make comparison easy:

1) Overview — mission and role in scholarships/mobility.  
2) Programs and services — recent or flagship offers.  
3) Target students and reach — who they serve and where.  
4) Partnership fit with UNIC/UNIC Athens — collaboration angles.  
5) Contacts and sources — official links and published emails.  
6) Next steps — concrete follow-ups or asks.

Name files with clear slugs (e.g., `foundation-tempus-euroguidance.md`) and link them from `../index.md`.

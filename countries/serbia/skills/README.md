# Serbia-specific skills

This folder holds procedures and templates that are specific to Serbia recruiting work. Shared, cross-country skills remain at the top level of `/skills/`. For partner mapping, start with the global `/skills/agents-and-partner-ecosystem/` skill and then apply the `serbia-agents-and-partner-ecosystem` addendum in this folder.

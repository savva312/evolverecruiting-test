# NGOs and SGOs (Nigeria)

Capture Nigerian NGOs/SGOs that affect scholarships, mobility, or career readiness. Use this space to prioritize outreach targets and store structured profiles.

- **Index:** See `index.md` for the prioritized list with links to profiles and statuses.
- **Profiles:** Create files under `profiles/` following the structure below.
- **Data:** Keep `../../data/entities/ngos-sgos.csv` aligned with the same organizations, statuses, and focus areas.

## Profile structure (standard)
Use consistent sections for every organization:
1. Overview
2. Programs and funding mechanisms
3. Target students and reach
4. Partnership fit for UNIC / UNIC Athens
5. Contacts and sources
6. Next steps

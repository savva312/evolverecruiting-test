# Nigeria reports

Use this folder for Nigeria-specific reports and synthesized outputs (e.g., channel reviews, university positioning summaries, fair calendars). Keeping reports here ensures country work remains isolated and easy to navigate.

## Usage
- Save each report as markdown unless a ticket explicitly requires another format.
- Name files descriptively (e.g., `digital-channels-2025.md`, `medicine-competitors.md`).
- Link between reports and other Nigeria content with relative paths (e.g., `../market/<file>.md`).
- Do **not** place Nigeria reports under `/executions/`; that path remains for global run logs.

# Nigeria school outreach plan (90-day): Lagos → Abuja → Port Harcourt
Last updated: 2025-12-22 | Owner: Nigeria Schools Partnerships Lead (TBD)

## Goal (90 days)

Build repeatable, city-by-city school partnerships that generate **qualified, consented leads** for **UNIC Nicosia** and **UNIC Athens**, then convert them into applications through a tight follow-up and handoff cadence.

This plan is designed for a **Jan–Mar execution window** (best for in-person outreach). If you run it in other months, use the calendar guardrails below to shift in-person weeks vs. virtual weeks.

## Inputs (source of truth)

- Target list + tiers (Tier1/Tier2): `../../entities/schools/index.md`
- Structured roster (use `school_id` as the key): `../../data/entities/schools.csv`
- Calendar constraints (avoid WAEC/NECO + end-of-term exams): `../../market/exams-and-calendar.md`

## Segmentation (how we prioritize)

### Tier1 (high priority)
Typical signals: hard-currency tuition, established international curricula (IB/AP/A-Level/French Bac), strong outbound orientation.

**Primary objective:** lock counselor partnership + referral flow (repeatable pipeline).

- Preferred motion: **Counselor briefing → student session → parent evening**
- Target cohorts: SS2 + SS3 (and equivalent Year 11–13); include “grade improvement” pathways (WAEC/NECO re-sits) where applicable.

### Tier2 (growth priority)
Typical signals: premium private schools, international exam signaling, mixed affordability, often less structured outbound counseling.

**Primary objective:** create demand through student/parent activation + “how to apply” clarity, then nurture.

- Preferred motion: **Student session + parent evening** (counselor briefing lightweight)

## Nigeria calendar guardrails (do/don’t)

Use month-level rules from `../../market/exams-and-calendar.md`:

- **Best for in-person school visits:** **late Jan–Mar** and **Sept–Oct**
- **Avoid heavy travel:** **May–July** (WAEC/NECO) and **mid–late Nov** (1st-term exams)
- **Use virtual-heavy play:** **Aug** (long vacation) and **Dec** (holiday break)

If you’re forced to operate during exam windows, keep presence via:
- counselor briefings (virtual)
- parent webinars (evenings/weekends)
- “results upload + next steps” clinics after WAEC/NECO result release windows

## Operating cadence (non-negotiable)

### Weekly rhythm (30–45 mins each)
- **Mon:** pipeline + calendar review (targets, meetings booked, upcoming sessions, risks)
- **Wed:** conversion standup (lead aging, Day 0/2/7/14 compliance, handoffs)
- **Fri:** quality control (tracking/UTMs, notes complete, next-week schedule locked)

### Service level agreements (SLAs)
- New leads from school events: **ingested + assigned within 24 hours**
- First outbound to new lead (email/WhatsApp/phone per consent): **within 48 hours**
- Counselor follow-up after any visit: **within 2 business days**

## 90-day plan (week-by-week)

This plan sequences **Lagos → Abuja → Port Harcourt**, while keeping follow-up running continuously for prior-city leads.

### Week 1–4: Lagos (build volume + establish Tier1 partnerships)
**Targets:** all Lagos Tier1 + Tier2A schools listed below.

- **Week 1 (Prep + booking)**
  - Clean target list from `schools.csv` and assign meeting “asks” per school tier.
  - Send counselor intro email + propose 2 dates (student session + counselor briefing).
  - Build event links/QRs with city + school attribution (see Measurement).
- **Week 2 (On-the-ground visits)**
  - Run **2–3 schools/day** (cluster by corridor: VI/Ikoyi/Lekki → Ajah/Eko Atlantic).
  - Execute at least: 2 counselor briefings, 2 student sessions, 1 parent evening.
- **Week 3 (Parent activation + conversion)**
  - One flagship **parent evening** (host school) + one virtual “study in Cyprus/Greece” webinar.
  - Admissions runs 1:1 calls for highest-intent leads; schools lead updates counselors.
- **Week 4 (Close + handoff)**
  - Apply Day 0/2/7/14 follow-up to every lead; escalate aged leads.
  - Counselor debrief email: “what we heard + next steps + next visit date window”.

### Week 5–8: Abuja (high-intent counselor pipeline)
**Targets:** Abuja Tier1 + Tier2A/Tier2B listed below.

- **Week 5 (Prep + booking)**
  - Lock counselor meetings early (Abuja calendars fill fast).
  - Prioritize schools with **USD/hard-currency tuition** and strong counseling signals.
- **Week 6 (On-the-ground visits)**
  - Focus on counselor briefings + smaller high-quality student groups.
  - Add one **counselor roundtable breakfast** (invite 6–10 counselors across Tier1/Tier2).
- **Week 7 (Parent activation + conversion)**
  - Parent evening (or virtual) focused on outcomes + process clarity (documents, timelines).
  - Admissions executes a “48-hour contact sprint” for every event lead.
- **Week 8 (Close + handoff)**
  - Audit lead records for consent + attribution; fix gaps while memory is fresh.
  - Confirm next-term touchpoint: “Counselor hour” date or next visit window.

### Week 9–12: Port Harcourt (targeted premium conversion)
**Targets:** Port Harcourt Tier2A/Tier2B listed below (Tier3 optional).

- **Week 9 (Prep + booking)**
  - Use a tighter set of asks: one student session + one parent evening anchored at a Tier2A school.
- **Week 10 (On-the-ground visits)**
  - Run clustered school stops + one evening parent event (or Saturday morning).
- **Week 11 (Conversion + scholarship/application clinic)**
  - Host an “application clinic” webinar for Port Harcourt leads (documents + next steps).
- **Week 12 (Close + handoff)**
  - Summarize results and lock the next 90-day cycle (expand to other cities via the roster).

### Days 85–90: Nigeria-wide conversion sprint
- “No lead left behind” review: every lead has a next action date or is marked closed-lost with a reason code.
- Run one national virtual session (counselors + parents) and invite Lagos/Abuja/PH cohorts.

## Core outreach motions (minimum 3)

Use `visit-templates.md` as the base format and standardize what “good” looks like.

### 1) Counselor briefing (45–60 mins)
**Goal:** earn counselor trust, agree referral process, lock dates for student/parent sessions.

- Agenda: UNIC + Athens overview → programs most relevant → admissions timeline → scholarships (guardrails) → Q&A → next steps
- Minimum “asks”:
  - permission to run a student session (date + venue)
  - counselor referral mechanism (link/form) + expected cohort size
  - approval for a parent evening/webinar invite through the school channel
- CTA + lead capture:
  - counselor gets a short “referral link” (UTM-tagged) and a 1-pager for parents

### 2) Student session (45 mins + Q&A)
**Goal:** generate opt-in leads and move students to a next step (webinar, 1:1, application starter).

- Format: 5-min “why UNIC/why Athens” → 15-min programs + outcomes → 10-min cost/scholarships guardrails → 10-min application steps → Q&A
- CTA options (choose one primary, one secondary):
  - Primary: “Scan QR to register for the next Nigeria webinar + get program guide”
  - Secondary: “Book a 10-minute eligibility check call”
- Lead capture rules:
  - Use a short form with consent; for minors, include a parent/guardian contact field and consent language (no WhatsApp without explicit opt-in).

### 3) Parent evening (60–90 mins)
**Goal:** convert intent into applications by addressing safety, value, recognition, and process clarity.

- Format: outcomes + student support → costs + scholarships guardrails → visa/process overview (no promises) → parent Q&A
- CTA:
  - “Start an application” + “Upload documents checklist” + “Book a parent call”
- Conversion mechanism:
  - Admissions counselor present (live or virtual) to schedule 1:1s on the spot.

## Follow-up cadence (Day 0 / 2 / 7 / 14)

Run this for every lead from school sessions and parent events.

- **Day 0 (same day):** thank-you message + program guide + next webinar date + “book a call” link
- **Day 2:** personalized nudge based on declared interest (program + campus) + reminder of required documents
- **Day 7:** invite to an application clinic (documents + deadlines) + counselor/parent-focused FAQ
- **Day 14:** “final nudge” + offer a quick eligibility check; if no response, move to low-frequency nurture

## Handoff to admissions (rule)

To keep speed without spamming, hand off a lead to the admissions pod when **any** of these are true:

- Completed lead form with consent **and** selected an intake window (or “apply now” intent), or
- Requested a call/appointment, or
- Uploaded documents / started an application, or
- Was referred directly by a counselor (explicit referral channel)

**Handoff SLA:** assign within 24 hours; first contact within 48 hours.

## Measurement (what we track)

Minimum outcomes per city sprint:
- meetings held (counselor briefings)
- student sessions delivered
- parent events delivered
- leads captured (consented)
- qualified leads handed to admissions
- applications started + submitted (tracked by admissions)

Attribution rules:
- Every school touchpoint uses `school_id` from `schools.csv` in the campaign/UTM naming.
- Keep a single source of truth (CRM preferred; otherwise a tracker) with **no PII stored in-repo**.

## Compliance and quality guardrails

- Do not promise visas, scholarships, recognition/licensure outcomes, or employment.
- Use role-based contacts only in documentation; store personal contact details only in approved systems (CRM).
- For minors: ensure consent language and parent/guardian contact collection where required; avoid retargeting practices that violate policy.

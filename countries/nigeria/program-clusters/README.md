# Program clusters

This area groups recruiting research and playbooks by program type so we can compare competitors and tailor Nigeria outreach for UNIC and UNIC Athens.

## Directory map
- [Medicine / MD](medicine-md/)
- [Health, Pharmacy, and Allied Health](health-pharmacy-allied-health/)
- [Computer Science, Data, AI, and Cyber](cs-data-ai-cyber/)
- [Business, Finance, Accounting, and Marketing](business-finance-accounting-marketing/)
- [Psychology, Counselling, and Behavioral](psychology-counselling-behavioral/)
- [Law, Governance, and International](law-governance-international/)
- [Design and Creative Media](design-creative-media/)

Each cluster folder contains:
- `cluster.md` for a brief description of the program family and what makes it attractive.
- `competitors.md` listing likely universities and private providers targeting Nigerian students.
- `nigeria-playbook.md` outlining outreach angles, channels, and partners to test.

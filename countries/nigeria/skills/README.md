# Nigeria-specific skills

Use this folder for procedures that only apply to Nigeria work. Each skill should live in its own subfolder with a `SKILL.md` that follows the repository’s skill format (YAML frontmatter + clear steps). Keep global, cross-country skills in the root `/skills/` directory. For partner mapping, start with the global `/skills/agents-and-partner-ecosystem/` skill and then apply the `nigeria-agents-and-partner-ecosystem` addendum here.

Available skills:
- `nigeria-repo-routing` — guidance on where to place Nigeria content and reports. See `nigeria-repo-routing/SKILL.md`.
- `nigeria-agents-and-partner-ecosystem` (Nigeria addendum; relies on the global agents-and-partner-ecosystem skill)
- `nigeria-competitive-landscape-by-program-cluster`
- `nigeria-data-model-csvs-and-profiles` (addendum to the global data-model Skill)
- `nigeria-education-system-and-calendar`
- `nigeria-paid-media-and-channel-playbooks`
- `nigeria-school-outreach-and-counselor-engagement` (addendum to the global school outreach skill)
- `nigeria-events-and-fairs` (addendum to the global events and fairs skill)
- `nigeria-program-clusters-and-competitive-sets`
- `nigeria-recognition-and-licensure`
- `nigeria-schools-and-counselor-mapping`
- `nigeria-schools-and-feeders`
- `nigeria-unic-portfolio-pricing-and-language`
- `nigeria-scholarships-and-discounts` (addendum to the global scholarships and discounts skill)

- `nigeria-offerholder-and-yield-playbooks` (addendum to the global offerholder/yield skill)

# Nigeria agents and partner ecosystem (addendum)

Use `/skills/agents-and-partner-ecosystem/SKILL.md` as the **primary** procedure for partner mapping. Apply this addendum only for Nigeria-specific nuances and paths.

## How to use

- Follow the global workflow for taxonomy, CSV schema, templates, and privacy rules.
- Layer in the Nigeria-specific emphases below when executing tickets scoped to `countries/nigeria/**`.

## Nigeria-specific partner emphases

- **Partner types to prioritize:** generalist agencies with Lagos/Abuja coverage; medicine-focused agencies (often bundling premed or interview prep); fair organizers running multi-city tours; counseling shops embedded in top private/boarding schools; diaspora or youth/community organizations that influence outbound choices.
- **City tags:** start with `Lagos`, `Abuja`, `Port Harcourt`, `Kano`, `Enugu`; add other hubs if the partner claims coverage. Note whether interactions are in-person offices or virtual-only.
- **Program-fit tags:** default to `medicine`, `cs-data-ai`, `business`; add `stem-engineering`, `law`, or `arts-media` only when evidence shows focus areas.
- **Engagement cues:** capture payment/commission preferences (e.g., wire vs. digital wallets), willingness to co-host webinars, and any requests for WA/Telegram follow-up while staying within data-handling rules.
- **Compliance reminders:** confirm data-consent practices and transparency on student fees; flag if partners expect cash incentives or rebates.

## Recommended Nigeria file paths

Use these when permitted by the ticket:

- `countries/nigeria/market/partners/index.md`
- `countries/nigeria/market/partners/profiles/<partner_slug>.md`
- `/countries/nigeria/data/partners/nigeria-partners.csv`

## Quick CSV and profile notes

- Keep `partner_id` slugs consistent with `city-keyword` patterns (e.g., `lagos-education-fair`, `abuja-premed-agency`).
- In profiles, include whether the partner can run hybrid/virtual events for non-Lagos audiences and note Nigeria-specific collateral needs (fees, recognition, payment options).

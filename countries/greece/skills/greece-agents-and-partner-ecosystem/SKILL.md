# Greece agents and partner ecosystem (addendum)

Use `/skills/agents-and-partner-ecosystem/SKILL.md` as the **primary** procedure for partner mapping. This file only records Greece-specific nuances and file locations.

## How to use

- Run the global workflow for taxonomy, CSV schema, templates, and privacy rules.
- Apply the Greece-specific emphases below when executing tickets scoped to `countries/greece/**`.

## Greece-specific partner emphases

- **Partner types to prioritize:** generalist agencies with Athens and Thessaloniki presence; medicine-focused consultancies (especially those advertising Cyprus placements); fair organizers covering Panhellenic school circuits; diaspora/Greek-community education groups that influence outbound choices; counseling hubs within top private schools or international programs.
- **City tags:** start with `Athens`, `Thessaloniki`, `Patras`, `Heraklion`; add islands/regions if a partner markets coverage there.
- **Program-fit tags:** default to `medicine`, `cs-data-ai`, `business`, and `psychology`; add `shipping-maritime` or `hospitality` where a partner demonstrably serves those segments.
- **Engagement cues:** note whether partners promote UNIC Nicosia versus UNIC Athens distinctly; capture their willingness for dual-campus positioning and Greek-language collateral needs.
- **Compliance reminders:** confirm GDPR and fee transparency; flag if partners require paper signatures or bank-transfer commission flows that affect onboarding.

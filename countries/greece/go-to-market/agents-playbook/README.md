# Education agents playbook (Greece)

Guidance for sourcing, onboarding, and managing agency partners who recruit for UNIC and UNIC Athens.

## Objectives
- Build a vetted roster of high-quality agents with clear territories and program focus.
- Set expectations on compliance, SLAs, and data sharing from day one.
- Drive consistent performance through training, marketing support, and quarterly reviews.

## Key actions
- Source candidates via associations, referrals, and target lists; pre-screen for sector focus and licenses.
- Run due diligence (references, license checks) before contracting; assign territories and caps where needed.
- Standardize onboarding: toolkit delivery, portal/CRM access, consent language, and fee schedules.
- Provide monthly enablement (program refreshers, objection handling, scholarship updates) with attendance tracked.
- Monitor performance and quality: application completeness, conversion by stage, and compliance adherence.

## Owners
- Country recruitment lead — approves agent selection, targets, and term adjustments.
- Agency manager/partnerships lead — runs onboarding, trainings, and performance reviews.
- Compliance/finance support — contract storage, invoice validation, and fee reconciliation.

## KPIs
- # of active agents signed vs. target; time from first contact to contract.
- Application quality score: % of submissions complete on first pass and with required documents.
- Conversion per agent (inquiry → offer → deposit) and yield vs. forecasts.
- SLA adherence: response time to inquiries, turnaround on document fixes, and data sync completeness.

## Starter checklists

**New agent onboarding checklist**
- Execute contract and store centrally with territory/program notes.
- Create CRM/portal access; share UTM/lead tagging rules for campaign attribution.
- Deliver onboarding deck + pricing/scholarship sheet; schedule first training within 7 days.
- Verify compliance acknowledgments (data protection, consent scripts) and ID a main point of contact.

**Monthly performance review checklist**
- Pull metrics by agent: volume, quality, conversion, and pending actions.
- Send scorecard and invite for review call; capture action items with owners/dates.
- Update enablement plan (webinars, new assets) based on gaps; confirm next milestone.

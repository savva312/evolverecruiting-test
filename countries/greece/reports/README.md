# Greece reports

Store Greece-only reports here (market scans, compliance memos, operational reviews, campaign retros). Keep each report in its own Markdown file with a clear filename so it can be cited elsewhere in `countries/greece/**`.

## Report format

- Title block with summary, owner, and `last_updated: YYYY-MM-DD`.
- Scope (what the report covers) and primary sources.
- Key findings and recommended actions for UNIC / UNIC Athens.
- Links to underlying data or annexes (place annexes in the same folder).

## Naming

Use `YYYYMMDD-topic.md` (e.g., `20251220-athens-market-scan.md`) to keep reports sortable by date.

## Folder layout

Keep this folder flat: place reports directly under `reports/` and keep any annexes alongside the parent file.

## What not to include

- Non-text binaries without a text extract.
- Bulgaria- or Nigeria-only content (keep those in their own country trees when available).
- Ticket management notes; put run notes in `/executions/` if needed.

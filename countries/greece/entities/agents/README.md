# Agents

Capture Greece-based education agents and counseling firms that can support UNIC recruiting. Track ownership details, portfolio focus, commission expectations, and performance notes with clear **UNIC Nicosia vs UNIC Athens** positioning.

- [`feeder-candidates.md`](feeder-candidates.md) — tiered roster with campus fit, program strengths, target segments, and relationship notes.
- `profiles/` — individual agent dossiers with ownership, destinations, commission norms, compliance checks, and campus-specific fit.

Use the roster as the entry point, then expand/refresh profiles as engagements progress. Keep evidence cited or mark assumptions explicitly when data is inferred.

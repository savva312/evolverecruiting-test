# NGOs and SGOs

Catalog non-governmental and semi-governmental organizations that influence scholarships, mobility support, or counseling ecosystems in Greece.

- [Index](index.md) — prioritized view of NGOs/SGOs, focus areas, and partnership status.
- `profiles/` — organization-specific notes with scope, offerings, and contact references. Each profile should cover: overview, programs/funding, target students, partnership fit, contacts/sources, and recommended next steps.

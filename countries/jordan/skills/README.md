# Jordan-specific skills

This folder holds procedures and templates that are specific to Jordan recruiting work. Shared, cross-country skills remain at the top level of `/skills/`. For partner mapping, start with the global `/skills/agents-and-partner-ecosystem/` skill and then apply the `jordan-agents-and-partner-ecosystem` addendum in this folder.

Available skills:
- `jordan-agents-and-partner-ecosystem` (Jordan addendum; relies on the global agents-and-partner-ecosystem skill)
- `jordan-competitive-landscape-by-program-cluster`
- `jordan-data-model-csvs-and-profiles` (addendum to the global data-model Skill)
- `jordan-education-system-and-calendar`
- `jordan-paid-media-and-channel-playbooks`
- `jordan-school-outreach-and-counselor-engagement` (addendum to the global school outreach skill)
- `jordan-events-and-fairs` (addendum to the global events and fairs skill)
- `jordan-program-clusters-and-competitive-sets`
- `jordan-recognition-and-licensure`
- `jordan-scholarships-and-discounts` (addendum to the global scholarships and discounts skill)
- `jordan-digital-measurement-and-benchmarks` (addendum to the global measurement skill)
- `jordan-schools-and-counselor-mapping`
- `jordan-schools-and-feeders`
- `jordan-unic-portfolio-pricing-and-language`
- `jordan-offerholder-and-yield-playbooks` (addendum to the global offerholder/yield skill)

# NGO/SGO profiles

Create one file per organization using the following structure:

1) **Overview** — mission, legal form, and why it matters for student mobility or funding.
2) **Programs and reach** — scholarships, training, counseling, internship/placement or refugee support, with geography and annual scale if available.
3) **Target students** — segments served (e.g., low-income Jordanians, refugees, STEM talent, orphans, youth employability).
4) **Partnership fit for UNIC/UNIC Athens** — how we could work together (referrals, co-branded scholarships, bootcamps, counselor training, events).
5) **Contacts and sources** — named contacts if public, URLs to official pages/press, and any diligence notes.
6) **Next steps** — concrete follow-ups (intro outreach, event invites, pilot proposal).

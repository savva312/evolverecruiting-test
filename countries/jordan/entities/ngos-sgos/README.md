# NGOs and SGOs

Catalog non-governmental and semi-governmental organizations that influence Jordanian student mobility, scholarships, and career services. Note focus areas, partnership fit, and sponsorship opportunities.

- `index.md` — prioritization view with quick links to profiles and partnership status.
- `profiles/` — one file per organization with mission, reach, programs, fit, contacts/sources, and next steps. Use the sections defined in this folder’s `profiles/README.md`.

## Tiering lens (aligned to skill)
- Ability to pay for **UNIC tuition ($10k–$15k) and Medicine ($22k–$27k)**, with published Grade 11–12 fees as the strongest signal.
- International curriculum/counseling readiness (IB DP, A-Levels, AP/US diploma).
- City/affluence proxy (West Amman dominates; Madaba’s King’s Academy functions as an Amman-market feeder).
- Access/readiness (evidence tags A = official fees, B = secondary listing, C = fees not public; verification queue noted below).

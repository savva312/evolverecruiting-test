# Competitors

Track universities and providers competing for Jordanian students. Summaries should cover positioning, pricing, scholarships, partnerships, and marketing tactics.

## Scope and priorities
- Focus on English- or bilingual-program providers that overlap with UNIC/UNIC Athens pipelines in STEM, business, law, health, and digital fields. Reference shared campus materials instead of duplicating them (e.g., [UNIC Athens programmes](../../../../UNIC/UNIC%20Athens/programs.md)).
- Prioritise institutions visible in Amman and major feeder governorates, plus nearby regional hubs (Cyprus, Turkey, EU) that market into Jordan.
- Capture any pricing/aid signals (tuition ranges, fee schedules, scholarship percentages) and channel tactics used for Jordanian recruitment.

## How to add profiles
- Create one Markdown file per competitor in `profiles/`.
- Include `last_verified`, 2–3 bullet sources, positioning vs UNIC/UNIC Athens, indicative pricing/fees/scholarships (with links), and any demand or channel signals observed in Jordan.
- Flag data gaps explicitly (e.g., “tuition not published—request from admissions”) to guide follow-up research.

- `profiles/` – competitor snapshots with differentiation notes and observed messaging.

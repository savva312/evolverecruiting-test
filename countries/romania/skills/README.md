# Romania-specific skills

This folder holds procedures and templates that are specific to Romania recruiting work. Shared, cross-country skills remain at the top level of `/skills/`. For partner mapping, start with the global `/skills/agents-and-partner-ecosystem/` skill and then apply the `romania-agents-and-partner-ecosystem` addendum in this folder.

Available skills:
- `romania-agents-and-partner-ecosystem` (Romania addendum; relies on the global agents-and-partner-ecosystem skill)
- `romania-competitive-landscape-by-program-cluster`
- `romania-data-model-csvs-and-profiles` (addendum to the global data-model Skill)
- `romania-education-system-and-calendar`
- `romania-paid-media-and-channel-playbooks`
- `romania-school-outreach-and-counselor-engagement` (addendum to the global school outreach skill)
- `romania-events-and-fairs` (addendum to the global events and fairs skill)
- `romania-program-clusters-and-competitive-sets`
- `romania-recognition-and-licensure`
- `romania-scholarships-and-discounts` (addendum to the global scholarships and discounts skill)
- `romania-digital-measurement-and-benchmarks` (addendum to the global measurement skill)
- `romania-schools-and-counselor-mapping`
- `romania-schools-and-feeders`
- `romania-unic-portfolio-pricing-and-language`
- `romania-offerholder-and-yield-playbooks` (addendum to the global offerholder/yield skill)

## 1. Intake & target validation
- [ ] Opportunity logged in CRM with `RO-AGT` tag and agent name.  
- [ ] Coverage confirmed (cities, channels, modal volumes).  
- [ ] Internal kickoff held (Partnerships + Agent Success + Admissions).  

## 2. Due diligence evidence

### Legal & ownership
- [ ] ONRC *certificat constatator* (≤30 days old) stored; CAEN codes align with education counseling.  
- [ ] VAT certificate / fiscal ID captured.  
- [ ] Beneficial owners vetted (sanctions + PEP check).  
- [ ] Risk rating documented (Low / Medium / High) with justification.  

### Financial & contracting
- [ ] Last two fiscal statements or accountant attestation reviewed.  
- [ ] Bank details (IBAN, SWIFT, beneficiary) verified and match legal entity.  
- [ ] Commission + marketing fees approved by Finance (note approver + date).  

### Sub-agent governance
- [ ] Written sub-agent policy received.  
- [ ] Current sub-agent list logged (names, geos, specialties).  
- [ ] Oversight evidence (contracts, QA schedule) reviewed.  

### GDPR & data handling
- [ ] Privacy notice + consent form reviewed; includes explicit consent for sharing data with UNIC.  
- [ ] Data storage location confirmed (EU/adequate jurisdiction).  
- [ ] Data Processing Agreement drafted / signed.  
- [ ] Lead capture workflow documented (form fields, consent proof, retention window).  

### Marketing claims & compliance
- [ ] Agent acknowledged use of approved program facts (`UNIC/unicnicosia/programs.md`, `UNIC/unicathens/programs.md`).  
- [ ] Deposit/visa timelines confirmed via `countries/romania/go-to-market/offerholder-and-yield/deposit-deadlines.md`.  
- [ ] Scholarship/discount messaging reviewed; “no guarantee” clause present.  
- [ ] Escalations or red flags noted (attach evidence + mitigation).  

**Compliance sign-off:** Name / Date

## 3. Contracting
- [ ] Latest agreement template populated with correct commission tiers.  
- [ ] GDPR + sub-agent clauses validated by Legal.  
- [ ] Agreement sent via DocuSign (date).  
- [ ] Fully executed contract stored (link).  
- [ ] Finance vendor setup completed (entity ID).  

**Legal sign-off:** Name / Date  
**Finance sign-off:** Name / Date

## 4. Training & enablement
- [ ] Product deep-dive delivered (UNIC Nicosia & Athens).  
- [ ] Cycle/deadline & compliance session delivered (exam calendar + deposit policy).  
- [ ] Counselor toolkit walkthrough completed (`countries/romania/go-to-market/schools-playbook/counselor-toolkit.md`).  
- [ ] Digital & lead capture training completed (`countries/romania/go-to-market/digital-playbook/channel-strategy.md`).  
- [ ] CRM credentials issued; test lead submitted.  
- [ ] Agent acknowledged receipt of full asset kit (email link).  

**Training owner sign-off:** Name / Date

## 5. Activation controls
- [ ] Campaign brief approved (channel mix, budget, start date).  
- [ ] Landing pages / lead forms reviewed for GDPR copy.  
- [ ] CRM automations configured (assignment rules, alerts).  
- [ ] First event or campaign scheduled (date/time).  

## 6. First 10 leads QA
- [ ] QA log created (link to tracker).  
- [ ] Batch 1 (leads 1–5) reviewed — result: ____ /5 green.  
- [ ] Batch 2 (leads 6–10) reviewed — result: ____ /5 green.  
- [ ] Corrective actions documented for any Amber/Red findings.  
- [ ] Go/No-go recommendation submitted to Partnerships Director.  

**Admissions QA sign-off:** Name / Date  
**Partnerships Director decision:** Scale / Slow / Pause (Date)

## 7. Post-onboarding handoff
- [ ] Partner added to `countries/romania/go-to-market/agents-playbook/partner-slas.md` with targets.  
- [ ] Reporting cadence agreed (weekly / bi-weekly).  
- [ ] Next quarterly review date booked.

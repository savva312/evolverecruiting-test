# NGOs and SGOs

Catalog non-governmental and semi-governmental organizations that support Romanian student mobility, scholarships, or career services. Note focus areas, partnership fit, and sponsorship opportunities.

- `index.md` — snapshot of priority organizations and links to full profiles.
- `profiles/` — one file per organization with mission, reach, partnership fit, contacts/sources, and next steps.
- Data lives in `../../data/entities/ngos-sgos.csv` and should mirror the profiles.

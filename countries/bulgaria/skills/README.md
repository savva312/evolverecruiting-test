# Bulgaria-specific skills

This folder holds procedures and templates that are specific to Bulgaria recruiting work. Shared, cross-country skills remain at the top level of `/skills/`. For partner mapping, start with the global `/skills/agents-and-partner-ecosystem/` skill and then layer on the Bulgaria addendum in this folder.

Available skills:
- `bulgaria-agents-and-partner-ecosystem` (Bulgaria addendum; relies on the global agents-and-partner-ecosystem skill)
- `bulgaria-competitive-landscape-by-program-cluster`
- `bulgaria-data-model-csvs-and-profiles` (addendum to the global data-model Skill)
- `bulgaria-education-system-and-calendar`
- `bulgaria-paid-media-and-channel-playbooks`
- `bulgaria-school-outreach-and-counselor-engagement` (addendum to the global school outreach skill)
- `bulgaria-events-and-fairs` (addendum to the global events and fairs skill)
- `bulgaria-program-clusters-and-competitive-sets`
- `bulgaria-recognition-and-licensure`
- `bulgaria-scholarships-and-discounts` (addendum to the global scholarships and discounts skill)
- `bulgaria-digital-measurement-and-benchmarks` (addendum to the global measurement skill)
- `bulgaria-schools-and-counselor-mapping`
- `bulgaria-schools-and-feeders`
- `bulgaria-offerholder-and-yield-playbooks` (addendum to the global offerholder/yield skill)

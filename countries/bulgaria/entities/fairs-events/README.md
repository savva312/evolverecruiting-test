# Fairs and events

List recruitment fairs, counselor gatherings, school tours, and community events relevant to Bulgarian students. Capture organizer details, audience size, pricing, and past performance.

- Planning view: `calendar-2025-2026.md` (priorities, timing windows, actions).
- Event one-pagers: `profiles/` (logistics, positioning, follow-up).
- Structured dataset: `../../data/entities/fairs-events.csv` (operational inventory for filtering, assignments, and CRM setup).
- Data dictionary: `../../data/entities/fairs-events-dictionary.md` (schema + rules for unconfirmed dates).

# NGOs and SGOs

Catalog non-governmental and semi-governmental organizations that support Bulgarian student mobility, scholarships, or career services. Note focus areas, partnership fit, and sponsorship opportunities.

- `index.md` – priority view with links to profiles, focus areas, and partnership status.
- `profiles/` – one file per organization following the structure: overview, programs, target students, partnership fit, contact/sources, next steps.
- Data: `../../data/entities/ngos-sgos.csv` tracks the same organizations in tabular form.

# Bulgaria playbook — Medicine / MD (operational v1)

This playbook turns the Bulgaria Medicine/MD cluster into an executable operating rhythm (schools + agents + digital + conversion).

Guardrails (non-negotiable):
- **No promises** on licensure, “automatic recognition”, residency placement, or employment outcomes.
- For “return-to-Bulgaria” questions, anchor to: `countries/bulgaria/market/recognition-and-licensure.md`.
- If unsure, use “We can explain the process and typical documents; the competent authority decides.”

Related references:
- Cluster: `countries/bulgaria/program-clusters/medicine-md/cluster.md`
- Competitors: `countries/bulgaria/program-clusters/medicine-md/competitors.md`

## 1) Targets and KPIs (what “good” looks like)
Track a minimum set weekly (by campus: Nicosia vs Athens, and by channel: schools/agents/digital/events):
- **Speed to lead:** median time-to-first-contact; target same-day.
- **Qualification rate:** % of leads that become “qualified MD leads”.
- **Application start rate:** % of qualified leads that start an application within 7 days.
- **Completion rate:** started → submitted.
- **Offer/acceptance:** submitted → offer; offer → deposit/commit (or equivalent “intent” step).
- **Cycle health:** # of qualified leads by intake month vs target.

Operational definition: a “qualified MD lead” is defined in `countries/bulgaria/program-clusters/medicine-md/cluster.md`.

## 2) Fast path: 7-day operational checklist (from qualified lead → application momentum)
Goal: within **7 days** of a lead qualifying, the candidate has a clear plan, correct expectations, and an application in motion.

### Day 0 (same day qualification is confirmed)
- Assign **single owner** (one admissions counselor accountable end-to-end).
- Send **MD triage pack** (high level): admissions steps + document checklist + compliance script on recognition/licensure.
- Book **2 touchpoints**: (1) admissions counseling call, (2) interview readiness / academic Q&A slot.
- Capture decision variables in CRM: campus preference, term, budget band, academic readiness, English readiness, decision-maker status.

### Days 1–2
- Run the **qualification + expectation** call:
  - confirm academics + English plan
  - confirm regulated-profession intent (Bulgaria / EU / elsewhere) and explain the difference between **academic recognition vs professional recognition**
  - agree the “application plan” timeline with parent/guardian present when possible
- If transfer candidate: start a **transfer pre-check** (what documents exist; what is missing; realistic credit expectations).

### Days 3–4
- Launch application support:
  - checklist the file (what is ready vs missing)
  - set a “next document due” date
  - book interview-prep slot with a defined agenda
- Share the “return-to-Bulgaria” expectations using `countries/bulgaria/market/recognition-and-licensure.md` (no outcome promises).

### Days 5–7
- Run **interview readiness** session (see Section 6).
- Confirm submission date and next steps (including deposit/commitment steps if applicable).
- If the family is stuck on cost/competitor comparisons, run a structured “decision meeting” (Section 7) instead of ad-hoc back-and-forth.

## 3) 90-day action plan (build pipeline + conversion)
This is the “doable in one quarter” plan to make Bulgaria MD recruiting operational.

### Days 1–30 — Foundation + launch
- Standardize a **single compliant script** for recognition/licensure questions using `countries/bulgaria/market/recognition-and-licensure.md`.
- Create a minimum **MD asset kit** (do not overclaim; keep it evidence-based):
  - “How admissions works” one-pager (steps + timeline)
  - “Return-to-Bulgaria” explainer (process + typical documents; no promises)
  - cost-of-attendance worksheet template (tuition + living + travel; ranges ok)
  - interview expectations overview (what we look for; what to prepare)
- Build a **school target list** by category (private/bilingual; science gymnasiums; high-output cities Sofia/Plovdiv/Varna/Burgas). Prioritize counselor relationships, not mass emailing.
- Stand up **webinar cadence** (2/month):
  - Webinar 1: “How to choose an English MD program (Bulgaria → EU)”
  - Webinar 2: “Recognition & licensure basics for returning to Bulgaria” (explicitly process-focused)
- Partner sprint: identify 10–15 likely feeder partners (agents + test-prep centers) and invite them to enablement.

### Days 31–60 — Scale channels
- Schools: run 6–10 counselor meetings and 3–6 in-school sessions (or virtual equivalents).
- Agents: onboard Tier 1 partners (Section 5), run monthly enablement, and enforce lead-quality standards.
- Digital: run 2–3 content pillars and retargeting (Section 4); measure by qualified leads, not raw leads.
- Implement “next best action” follow-ups: every qualified lead has a planned next step within 72 hours.

### Days 61–90 — Optimize conversion + build repeatability
- Review call recordings/notes for objection patterns; update scripts monthly.
- Add a monthly “offer-holder yield” clinic: admitted candidates + parents join Q&A on next steps and paperwork (no PII in group settings).
- Publish a cycle dashboard for Bulgaria MD (KPIs in Section 1) and identify the two biggest conversion leaks; fix them.

## 4) Channel plan (what to do, by channel)

### A) Schools & counselors (highest trust channel)
**Targets:**
- private/bilingual schools and science gymnasiums with high outbound mobility
- counselor networks and pre-med clubs

**Plays:**
- “Counselor briefing” (30 minutes): admissions timeline + candidate readiness + compliant recognition script
- “Family info session” (45 minutes): decision drivers + process + Q&A
- “Interview readiness workshop” (30 minutes): what we assess and how to prepare

**Non-negotiable:** always include the compliance disclaimer and point to `countries/bulgaria/market/recognition-and-licensure.md` for Bulgaria-specific return questions.

### B) Agents (volume + reach, requires discipline)
**Partner rules:**
- No partner is allowed to promise recognition/licensure outcomes.
- Use a single agreed intake form; reject low-information leads.

**What to measure:** qualified lead rate, application starts, file completeness at submission, and show-up rate to interviews.

### C) Test-prep centers (biology/chemistry + admissions coaching ecosystem)
**Why it works in Bulgaria:** many candidates are already in structured prep for CEE medical entrance exams.  
**Plays:** co-host “How admissions differs by country” sessions; offer counseling slots for their top students; create a shared calendar for application deadlines.

### D) Digital (demand capture + nurture)
**Always optimize for qualified leads**, not cheap leads.
- Search: “medicine in English”, “medical university abroad”, “medicine Cyprus”, “medicine Greece” (local-language variants should be tested by the team).
- Social: parent-focused and student-focused creative; lead magnets should be process-oriented (timeline/checklist), not outcome claims.
- Nurture: 5-touch sequence over 14 days (webinar invite → checklist → counselor call → interview prep → “return-to-Bulgaria” process explainer).

### E) Events/fairs
Use events selectively; treat them as **lead qualification machines**:
- capture minimum data to follow up within 24–48 hours
- schedule counseling calls at the booth when possible

## 5) Partner plan (agents + schools + community)
Create a simple tiering and enablement system.

### Tiering (example)
- **Tier 1 (core):** consistently sends qualified leads; follows compliance; accepts feedback.
- **Tier 2 (developing):** sends volume but needs training; quality inconsistent.
- **Tier 3 (opportunistic):** event-based or seasonal; do not invest heavy time.

### Enablement cadence
- Monthly 45-minute partner training (recorded) + monthly Q&A clinic.
- Quarterly compliance refresh: “what we can/can’t say” + sample scripts.

### Lead-quality gate (protect the funnel)
Reject or downgrade leads missing the minimum: academic profile, timeline, budget range, and regulated-profession intent (where they want to practise).

## 6) Interview pipeline + readiness (high level; no PII)
Goal: maximize interview show-up and reduce “avoidable” rejections due to preparation gaps.

### Readiness checklist (candidate)
- Motivation story: “why medicine” + “why now” + “why this campus” (specific but truthful)
- Evidence of learning habits and resilience (how they handle setbacks)
- Ethical reasoning and professionalism (no coaching to “say the right thing”; coach to be clear)
- Communication in English (structured practice plan if weak)

### Internal process
- Pre-interview call (15 minutes): expectations + logistics + compliance reminder (no outcome promises)
- Post-interview follow-up within 48 hours: next steps + what to prepare next (documents/deposit/deferral rules)

## 7) Objection handling (decision meeting structure)
Use a structured “decision meeting” when a family is stuck between competitors (especially Bulgarian publics and CEE options).

Agenda (20–30 minutes):
1) Confirm non-negotiables (budget ceiling, campus preference, term, language comfort).
2) Compare **total cost + risk** (not just tuition): admin friction, clinical language requirements, timeline predictability.
3) Re-center on regulated-profession reality and the process in `countries/bulgaria/market/recognition-and-licensure.md`.
4) Agree the next step within 72 hours (application start or scheduled follow-up).

## 8) Document checklist (high level; adapt per admissions rules)
Do not collect unnecessary personal data in shared folders; follow institutional privacy policy.

Typical checklist categories (examples; confirm current requirements each cycle):
- Identity document (for application identity verification)
- Academic records: transcripts and graduation evidence (or expected graduation date)
- English proficiency evidence (test scores) or an assessment plan
- Any required personal statement/motivation materials
- For transfer cases: transcripts + course descriptions/syllabi (for credit assessment)

For “return-to-Bulgaria” planning (early preparation, before graduation), point candidates to the “prepare early” lists in:
- `countries/bulgaria/market/recognition-and-licensure.md`

## 9) Competitor-linked outreach notes (channel tactics)
- **CEE entrance-exam ecosystem (Czech/Hungary):** partner with Sofia-based prep centers; offer a “second option” pathway with clear admissions steps and compliance-first recognition messaging.
- **Bulgarian public med schools (MU Sofia/Varna/Plovdiv):** run “value vs price” counseling using total-cost + risk framing; avoid negative claims about competitors.
- **Poland/Slovakia fairs:** attend only if you can follow up within 48 hours and book counseling calls during the event to prevent lead decay.

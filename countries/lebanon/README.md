# Lebanon country workspace

Use this README as the navigation hub for Lebanon-specific recruiting materials that support UNIC and UNIC Athens.

## Quick starts
- **Outbound mobility baseline:** [`2025-12-20 - Lebanon Outbound Mobility Baseline`](reports/2025-12-20%20-%20Lebanon%20Outbound%20Mobility%20Baseline.md)
- **UNIC Athens recruitment plan:** [`2025-12-20 - UNIC Athens Lebanon Recruitment Plan`](reports/2025-12-20%20-%20UNIC%20Athens%20Lebanon%20Recruitment%20Plan.md)
- **Data dictionary and field standards:** [`Data README`](data/README.md) and [`field-standards.md`](data/field-standards.md)

## Directory guide
- **market/** — Mobility, affordability, education system, and recognition/licensure research to size the Lebanese opportunity. Start with [`market/outbound-mobility.md`](market/outbound-mobility.md) and the education system and exams references.
- **entities/** — Profiles for schools, agents, regulators, NGOs/SGOs, fairs/events, and competitors to target and manage. See [`entities/README.md`](entities/README.md) for subfolder scopes.
- **go-to-market/** — Channel playbooks for schools, agents, digital, and offerholder/yield. Begin with the hub [`go-to-market/README.md`](go-to-market/README.md) and dive into the relevant playbook folder.
- **program-clusters/** — Program-specific playbooks and competitor scans. Use [`program-clusters/README.md`](program-clusters/README.md) to jump into clusters like medicine, CS/data/AI, and business.
- **data/** — Structured CSVs and dictionaries for Lebanon recruiting. Anchor on [`data/README.md`](data/README.md) and [`data/field-standards.md`](data/field-standards.md) before adding tables.
- **reports/** — Polished outputs and baselines. The outbound mobility baseline and UNIC Athens recruitment plan live here alongside other ready-to-share reports; see [`reports/README.md`](reports/README.md) for formatting expectations.
- **skills/** — Lebanon-only skill addenda aligned to global standards (e.g., agents, school outreach, events, data model). Start at [`skills/README.md`](skills/README.md) to find the right procedural playbook.

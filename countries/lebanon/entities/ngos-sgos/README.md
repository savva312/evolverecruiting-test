# NGOs and SGOs

Catalog non-governmental and semi-governmental organizations that support Lebanese student mobility, scholarships, or career services. Note focus areas, partnership fit, and sponsorship opportunities.

- **Start here:** [`index.md`](index.md) for prioritized organizations, scope/focus summaries, and quick links to profiles.
- **Profiles:** `profiles/*.md` — include overview, programs, target students, partnership fit, contacts/sources, and next steps. See [`profiles/README.md`](profiles/README.md) for the required sections.
- **Data:** Keep `../../data/entities/ngos-sgos.csv` in sync with the profiles and the [data dictionary](../../data/entities/ngos-sgos-dictionary.md); columns capture scope, focus area, audience, partnership notes, status, and last-reviewed date for each organization.

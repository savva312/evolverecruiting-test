# Lebanon school profiles hub

Use this area to catalog Lebanese secondary schools relevant for outreach. The current focus is on high-priority USD-feeder schools that align to UNIC tuition levels. Navigation:
- [Profiles index](profiles/README.md) for quick links to every high-priority school profile.
- Structured data in [`data/entities/schools.csv`](../../data/entities/schools.csv) (one row per high-priority school).

Structure:
- `profiles/` – city-specific folders for school summaries, contact details, and visit histories.
- Additional shared templates or contact logs can live alongside this README.

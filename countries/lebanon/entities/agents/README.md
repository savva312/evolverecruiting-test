# Agents

Capture Lebanese education agents and counseling firms that can support UNIC Nicosia and UNIC Athens. Include ownership details, credentials, commission expectations, FX handling, and performance notes.

- [`../../data/entities/agents.csv`](../../data/entities/agents.csv) — structured agent dataset (one row per profile).
- [`../../data/entities/agents-dictionary.md`](../../data/entities/agents-dictionary.md) — data dictionary for the CSV fields.
- [`feeder-candidates.md`](feeder-candidates.md) — tiered roster with campus fit, program focus, city coverage, and risk/credibility notes.
- `profiles/` — individual agent dossiers with contacts, office locations, service mix, vetting flags, and sources/assumptions.

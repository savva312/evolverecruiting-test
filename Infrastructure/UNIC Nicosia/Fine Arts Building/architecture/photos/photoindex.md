# Fine Arts Building (UNIC Nicosia) — photo index — 2025-12-22

- **Location:** UNIC Nicosia — Fine Arts Building
- **Photos:** 31 (`.jpeg`)
- **Evidence style:** “Observed” stays literal as much as possible; stakeholder notes explain how each photo can be used to make decisions.

## Scope of this set

Public frontage and entry sequence; side service lane (including external services); rear/loading functions; and adjacent gravel yard/perimeter conditions.

## How to use this index

- Use the linked filenames to open images; the index is ordered to follow a plausible walking/arrival path where possible.
- Treat photos as supporting evidence; confirm code-critical dimensions (accessibility, egress, clearances) with measurements/as-builts.
- When something is unclear, the safest next step is a targeted re-shoot: one wide establishing shot + one close-up of labels/controls/details.

## Stakeholder perspectives (what to look for)

### Architect / designer
- Read the portal/entry composition as an “address”: depth of recess, glazing strategy (diffuse/frosted light), and how landscape frames approach.
- Track where exposed services and conduits disrupt the façade rhythm; note opportunities for alignment/screening without losing maintainability.
- Compare public frontage vs rear/service elevations to ensure the building reads coherent from key approaches.

### Engineer (MEP / structural / civil)
- Use ramp/handrail/tactile shots to flag accessibility checks; confirm slopes, landings, and thresholds via measurement.
- Use yard/puddling shots to assess grading and stormwater capture; gravel areas often need engineered sub-base/drainage if heavily used.
- For loading bays/roller doors, verify apron drainage away from thresholds and safe vehicle maneuvering.

### Facilities / operations
- Use signage/waste/service-lane photos to map pickup routes, storage rules, and where housekeeping needs tightening.
- Yard photos are evidence for dust/mud and puddling; prioritize surface upgrades if the area is used year-round.
- Treat weathering/graffiti as high-leverage perception fixes (brand markers, boundary walls).

### Marketing / admissions
- Lead with clean frontage, portal, signage wall, and any lobby/gallery views; these sell “creative culture” best.
- Use service/yard shots sparingly (operations story, making logistics) unless the area is staged/cleaned.
- Build a future set focused on student work, studios, and exhibitions (with permissions).

### Educator / student experience
- Use lobby/gallery and entry sequence to explain where students meet, exhibit, and transition between making and display.
- Use workshop/service-zone evidence to reinforce safety culture (PPE, storage discipline, clear routes).
- Use accessibility features (tactile paving, ramps) in onboarding to model inclusive campus expectations.

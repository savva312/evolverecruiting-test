# RTB (Research and Technology Building)

This folder tracks facility documentation for the RTB (Research and Technology Building) at UNIC Nicosia.

Use the base scaffold from `skills/infrastructure-building-folders/SKILL.md`:
- `planning/` — campus planning, access, zoning, and accessibility notes tied to the building site.
- `architecture/` — scaled drawings and building-level imagery. Keep plans/exports in `architecture/floorplans/` and exterior/common-area photos in `architecture/photos/` using the photo documentation guidelines.
- `rooms/` — room-level documentation (purpose, capacity, equipment) once spaces are profiled.

Add further subfolders (engineering, operations, compliance, room/equipment details) only when real content is ready to keep the tree lean.

# UNIC Athens housing

Consolidate accommodation options and workflows for Athens-bound students.

- University-managed housing inventory, room types, pricing, and amenities.
- Booking rules (eligibility, deposits, contract timelines) and required documents.
- Trusted private housing options/partners, average costs, and neighborhood guidance.
- Move-in/check-in steps, key handover, and support contacts.
- Accessibility considerations and safety/commute tips.

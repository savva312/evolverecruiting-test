# UNIC Athens campus workspace

Use this folder as the shared source of truth for UNIC Athens campus materials. Keep country-level tailoring inside `/countries/<country>/`; maintain campus policies and assets here for reuse across markets.

## Folder map

- `programs/` — Athens program inventory and positioning.
- `admissions/` — entry standards, document formats, English requirements, and intake calendars.
- `visas/` — Greece study permit/visa requirements, biometrics locations, timelines, and sources.
- `housing/` — residence options, vetted partners, deposits/contracts, and check-in steps.
- `scholarships/` — scholarship/bursary rules, eligibility, stacking limits, and renewals.
- `student-life/` — orientation/onboarding, health insurance, arrival logistics, and campus services.

Note sources and “last verified” dates so downstream teams can trust the guidance.

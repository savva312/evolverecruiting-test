# UNIC Nicosia visas and immigration

Document the Cyprus study permit process for UNIC Nicosia applicants.

- Permit category, eligibility (work rights, dependents), and health/financial proofs.
- Document checklist (translations, notarization, apostille), biometrics locations, and medicals.
- Application steps (pre-arrival vs. on-arrival), expected timelines, and fees.
- Common failure points and escalation paths with admissions/intl student office contacts.
- Official sources (gov/university) with date stamps for verification.

# UNIC Nicosia campus workspace

Use this folder as the shared source of truth for campus-level recruiting content that every country can reference. Keep country-specific tactics inside each `/countries/<country>/` directory and store campus policies and assets here.

## Folder map

- `programs/` — campus program inventory and positioning.
- `admissions/` — entry criteria, document templates, English-language rules, and intake calendars.
- `visas/` — Cyprus study permit requirements, biometrics locations, timelines, and source links.
- `housing/` — university-managed dorms, vetted private options, booking rules, and check-in workflows.
- `scholarships/` — campus scholarship/discount policies, eligibility rules, and renewal criteria.
- `student-life/` — onboarding/orientation, health insurance, arrival support, and student services.

Add source links and “last verified” notes to every page so downstream country teams can trust the material.

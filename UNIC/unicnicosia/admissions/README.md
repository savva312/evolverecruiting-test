# UNIC Nicosia admissions

Capture campus-wide admissions criteria and process steps that apply across countries.

- Degree- and program-level entry requirements (academic subjects/grades, English proofs, interviews).
- Required documents with format and translation rules.
- Application channels, fee policies, and intake calendars.
- Evaluation timelines, decision SLAs, and communication templates.
- Source links and “last updated/verified” stamps.

# UNIC recruiting workspaces

This directory centralizes campus-specific recruiting content for the University of Nicosia system.

- [unicnicosia/](./unicnicosia/README.md) — UNIC Nicosia campus workspace covering programs, admissions, visas, housing, scholarships, and student life.
- [unicathens/](./unicathens/README.md) — UNIC Athens campus workspace with the same subfolders for Athens-specific guidance.
- [data/](./data/README.md) — shared datasets (e.g., consolidated program inventory) used by both campuses.

Folders use lowercase, URL-friendly names so links stay stable across countries. Use each campus folder to capture program inventory, positioning, and campus-level playbooks that countries can reference without duplicating content.

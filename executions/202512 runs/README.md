# /202512 runs/ (EvoBuilder run staging)

This folder is reserved for EvoBuilder runs executed in December 2025. Create one subfolder per run using the standard timestamp format:

- `YYYY-MM-DD_HHMMSS/` (e.g., `2025-12-20_141530/`)

Inside each run folder, include:
- `run_manifest.md` (run id, date/time, request, tickets, PR link, status, notes)
- run artifacts such as `evobuilder_run_plan.md`, `evobuilder_workplan.md`, `round_<n>_research.md`, and assembled outputs as needed.

Keep ticket deliverables in their ticket-scoped locations; `/executions/` is only for run-scoped artifacts.
